NLP Topic Modeling

This project is my Master’s Thesis, focusing on advanced Natural Language Processing (NLP) techniques applied to real-world healthcare communication data.

Links

  • GitHub Repository: View Project Code

Technologies Used

  • BERTopic
  • Latent Dirichlet Allocation (LDA)
  • Python
  • High-Performance Computing (HPC)

Project Objective

  • Extract and analyze latent topics and recurring themes from messages exchanged between patients and their assigned health coaches.
  • Context: In digital healthcare platforms, individuals with chronic conditions such as diabetes and obesity are paired with a coach. Through applications like the Liva platform, they communicate regularly to track progress, provide support, and improve health outcomes.
